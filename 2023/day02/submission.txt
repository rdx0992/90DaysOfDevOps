Task: What is the linux command to 
1. Check your present working directory.
Ans. for checking the current working directory use "pwd" 

2. List all the files or directories including hidden files.
Ans. for listing all the files or directories including hidden use "ls -a"

3. Create a nested directory A/B/C/D/E
Ans. for creating a nested directory A/B/C/D/E use "mkdir -p A/B/C/D/E"
